# 《苦力怕中立》模组介绍

## 🎮 模组简介
**《苦力怕中立》** 是一个为 Minecraft Java 版 1.20.1 Forge 平台设计的创新模组，旨在彻底改变游戏中原版苦力怕的敌对行为。通过本模组，苦力怕从原本的敌对生物转变为中立生物，为玩家带来全新的游戏体验和更多生存策略选择。

## 🔧 核心功能

### 🟢 中立化苦力怕
- **默认中立行为**：苦力怕不再主动攻击玩家，像牛、羊等友好生物一样在世界中自由活动
- **自然互动**：玩家可以安全地观察、靠近苦力怕，不会被无端攻击
- **生态融合**：苦力怕成为游戏世界生态的一部分，而非单纯的威胁

### ⚡ 条件性反击机制
- **攻击触发**：只有当玩家主动攻击苦力怕时，它才会进入敌对状态
- **智能追击**：被激怒的苦力怕会像原版一样追击攻击者
- **可控爆炸**：激怒后的爆炸行为可根据配置调整

## ⚙️ 默认配置
模组安装后即启用以下平衡设置：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 模组状态 | ✅ 启用 | 控制模组是否生效 |
| 主动攻击 | ❌ 禁用 | 苦力怕不会主动攻击 |
| 激怒爆炸 | ✅ 是 | 被攻击后苦力怕会爆炸 |
| 持续时间 | 20秒 | 激怒状态持续时间 |
| 需要潜行激怒 | ❌ 否 | 任何攻击都能激怒 |
| 爆炸破坏 | ❌ 否 | 爆炸不破坏地形 |

## 🎮 游戏体验

### 🟢 **和平共处模式**
- 在洞穴探索时不再担心突然的爆炸
- 建造基地时苦力怕成为无害的"邻居"
- 新手玩家可以更专注于建筑和探索
- 创造更有趣的农场和生物观察站

## 🛠️ 命令系统

### 📋 **查看状态**
```
/endermanneutral status
```
显示当前所有配置状态，帮助玩家了解模组运行情况。

### 🔄 **重新加载配置**
```
/endermanneutral reload
```
无需重启游戏，实时应用配置更改。

## 🌟 模组亮点

### 1. **颠覆性创新**
彻底改变苦力怕的游戏角色，从"恐怖分子"变为"中立居民"。

### 2. **高度可配置**
玩家可根据自己的游戏风格调整所有参数：
- 激怒持续时间：5-300秒可调
- 爆炸破坏控制：保护建筑完整性
- 潜行攻击要求：增加策略深度

### 3. **沉浸式体验**
苦力怕行为更加合理：
- 白天也会在户外活动
- 对玩家保持适当距离
- 被攻击后才有攻击性


## 📦 安装说明

### 环境要求
- **Minecraft版本**：1.20.1
- **模组加载器**：Forge 47.4.13
- **Java版本**：Java 17

### 安装步骤
1. 确保已安装正确版本的 Forge
2. 将模组文件放入 `.minecraft/mods` 文件夹
3. 启动游戏，选择 Forge 配置文件
